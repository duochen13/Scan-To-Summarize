# TextToAudioBeta

### frontend
```
pod install
open ListenText.xcworkspace 
```

### backend
```
cd server
source env/bin/activate
python3 index.py (localhost:5000)
```

### api endpoints
```
/: main page
/test: received post request from frontend

```
